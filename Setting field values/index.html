This example shows that you can not only read a value in a field with document.getElementById..., but can also "write" a value into a field.

In this case, the function assigns the value found in the ZIP field to a variable. Then using a switch statement, it matches the ZIP to a city name and assigns that name to a second variable. Then, using the variable, it places the city name in the city field.
