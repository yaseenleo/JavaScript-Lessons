Steps in While loops

  • var i = 0 establishes a counter.
  • i <= 3 says the loop is to keep running as long as i is less or equal to 3.Since i is assigned 0 initially \, the loop will run 4 times.
  • i++ increment i on each iteration

A while loop does the same thing, but it's organized differently. Only the middle term, how long to keep the loop going, is inside the parentheses. The counter is defined before the first line of the while block. The counter is updated within the code that executes when the loop runs.

In other ways a while loop is organized along th esame lines as a for loop

  • parentheses enclose the loop limiter.
  • Curly brackets enclose the code that executes.
  • The code that executes is indented 2 spaces.

Since any for loop can be translated into a while loop and vice versa, you can use whichever one you prefer.
