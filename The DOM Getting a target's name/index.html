It's possible to confuse the node value of a text node with the innerHTML property. There are two differences

  • innerHTML is a property of the element node, <h2>, for example. The node value is a property of the text node itself, not the parent element.
  • innerHTML includes all the descendants of the element, including any inner element nodes like as well as text nodes. The node value includes only the characters that comprise the single text node.
