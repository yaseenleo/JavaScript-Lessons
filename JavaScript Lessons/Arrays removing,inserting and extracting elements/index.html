Rules and Important points of Array removing,inserting and extracting elements

  1. Use the shift method to remove an element from the begining of an array.
  2. To add one or more elements to the beginning of an array, use the unshift method.
  3. Use the splice method to insert one or more elements anywhre in an array. while optionally removing one or more elements that come after it.
  4. Use the slice method to copy one or more consecutive elements in any position and put them into a new array.

  • Since the first index number inside the parentheses specifies the first element to be copied, you might think the second index number specifies the last element to be copied. In fact, the second number specifies the index number of the element after the last element to be copied.
  • You must assign the sliced elements to an array. It could, of course, be the same array from which you're doing the slicing. In that case, you'd be reducing the original array to only the copied elements.
