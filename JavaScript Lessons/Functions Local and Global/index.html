A global variable is one that's declared in the main body of your code, that is not inside a function

A local variable is one that's declared inside a function. It can be either a parameter of the function which is declared implicity by being named as a parameter, or a variable declared explicity in the function with the var keyword

Global variable Local variable
Declared in the main code Declared in a function
Known everywhere, useable everywhere Known only inside the function, useable only inside the function
